SetScreenMode

The SetScreenMode method can be used to hide the status bar and virtual menus and buttons.

 app.SetScreenMode( mode );

Use "Full" to hide the status bar or "Game" to hide the status bar and the virtual menus and buttons found on many modern devices.

Example

function OnStart()
{
  app.SetScreenMode( "Full" );
 }
